Frequently Asked Questions

1. What is the difference between var, let, and const?
Var is the old method to declare a variable. We don't use this anymore. Let is block scoped and we can change this later. But const is same as let expect it cannot change later .
2. What is the difference between map(), forEach(), and filter()?
We can use map() to change element (like-each element can multiply by 2). forEach() basically iterates over elements but does not return anything . filter() basically select elements based on a condition.
3. Explain arrow functions and how they are different from regular functions?
Arrow function is shorter than regular function. Arrow function return implicitly if we write it in one line.
4. How JavaScript Promises work?
Js promise work on 3 states. 1.Pending = Initial state, operation not completed. 2. Fulfilled = Operation completed.. 3. Rejected = Operation failed.
5. How closures work in JavaScript?
When a function is created inside another function, it creates a closure. The inner function remembers the variables of the outer function. Even after the outer function returns, the inner function still has access to those variables.
